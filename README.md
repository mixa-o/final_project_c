# Итоговый семестровый проект 

Модульная программа на языке Си, объединяющая лабораторные работы.

## Структура проекта
`main.c` — главный управляющий модуль.
`include/` — папка с заголовочными файлами (`.h`).
`src/` — папка с файлами исходного кода (`.c`).

## Реализованные модули 
1.  **Лабораторная 1**: Расчет пути торможения самолета при посадке.
2.  **Лабораторная 2**: Вычисление суммы первых N членов ряда.
3.  **Лабораторная 3**: Посимвольный анализ текста, счёт числа слов, заканчивающихся на согласную букву.
4.  **Лабораторная 4**: Работа со строками и файловым вводом/выводом, удаление предполсденего слова.
5.  **Лабораторная 5**: Обработка одномерных массивов, и анализ структуры массива памяти с помощью указателей.
6.  **Лабораторная 6**: Работа с двумерными массивами и их преобразование.
7.  **Лабораторная 7**: Поразраядная обработка целых чисел, побитовые операции и анализ байтов числа на арифметичскую прогрессию.

## Сборка и запуск
Для сборки проекта в ручном режиме через GCC используется команда:
```bash
gcc -o main main.c src/*.c -Iinclude
```
